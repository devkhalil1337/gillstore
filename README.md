# Mini-sol Backoffice

Technologies
- Nodejs
- Angularjs
- Bootstrap 4


Run
- Install <a href="https://www.npmjs.com/get-npm"  target="_blank">npm</a>
- Install all packages by running this command "npm install"
- nodemon server.js
- Click on <a href="http://localhost:5000" target="_blank">http://localhost:5000</a>
